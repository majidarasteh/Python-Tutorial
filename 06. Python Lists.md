# **Introduction to Python Lists**

A **list** in Python is a versatile and ordered collection of elements enclosed in square brackets `[ ]`. Lists can store items of **different data types**, including numbers, strings, and even other lists (nested lists). They are **mutable**, meaning elements can be added, removed, or modified after creation. Lists support **indexing, slicing, iteration, and a variety of built-in methods** that make them powerful for storing and manipulating data efficiently. Their flexibility and ease of use make lists one of the most commonly used data structures in Python programming.

# **1. Python Lists**

A **list** in Python is created using square brackets **[ ]** with elements separated by commas.

```python
my_list = [1, 2, 3]
print(my_list)
```

**Output**

```
[1, 2, 3]
```

### Key Properties

* A list can contain **any number** of elements.
* Elements may be of **mixed data types** (int, float, string, etc.).
* Lists can contain **nested lists**.
* Lists are **ordered** and maintain insertion sequence.
* List indexing starts at **0**.
* Lists are **mutable** (elements can be changed).
* Lists can hold **a wide range of data types**.

# **5. List Slicing**


# **7. List Methods with Explanations + Examples**

### **append() — adds one element to end**

```python
items = [1, 2, 3]
items.append(4)
print(items)
```

**Output**

```
[1, 2, 3, 4]
```

---

### **extend() — adds elements from another iterable**

```python
nums = [1, 2]
nums.extend([3, 4])
print(nums)
```

**Output**

```
[1, 2, 3, 4]
```

---

### **insert() — inserts at a specific index**

```python
letters = ["a", "b", "d"]
letters.insert(2, "c")
print(letters)
```

**Output**

```
['a', 'b', 'c', 'd']
```

---

### **pop() — removes and returns an element**

```python
stack = [10, 20, 30]
x = stack.pop()
print(stack, x)
```

**Output**

```
[10, 20] 30
```

---

### **remove() — removes first matching value**

```python
nums = [1, 2, 3, 2]
nums.remove(2)
print(nums)
```

**Output**

```
[1, 3, 2]
```

---

### **sort() — sorts list in place**

```python
nums = [5, 1, 4]
nums.sort()
print(nums)
```

**Output**

```
[1, 4, 5]
```

---

### **sorted() — returns a new sorted list**

```python
nums = [3, 1, 2]
print(sorted(nums))
print(nums)
```

**Output**

```
[1, 2, 3]
[3, 1, 2]
```

---

### **reverse() — reverses list order**

```python
nums = [1, 2, 3]
nums.reverse()
print(nums)
```

**Output**

```
[3, 2, 1]
```

---

### **copy() — creates a new independent list**

```python
x = [10, 20]
y = x.copy()
y.append(30)
print(x, y)
```

**Output**

```
[10, 20] [10, 20, 30]
```

---

### **count() — counts occurrences**

```python
nums = [1, 2, 2, 3]
print(nums.count(2))
```

**Output**

```
2
```

---

# **10. Looping, Membership, and List Comprehensions**

### Membership test: in / not in

```python
fruits = ["apple", "banana"]
print("apple" in fruits)
print("mango" not in fruits)
```

**Output**

```
True
True
```

### Looping examples

```python
# Example 1
for var in ['apple','banana','mango']: 
    print("I like", var)
```

**Output**

```
I like apple
I like banana
I like mango
```

```python
# Example 2 - Using enumerate
fruits = ['apple', 'banana', 'mango']
for index, item in enumerate(fruits):
    print(index, item)
```

**Output**

```
0 apple
1 banana
2 mango
```

### List comprehension

```python
nums = [1, 2, 3, 4]
squares = [n*n for n in nums]
print(squares)
```

**Output**

```
[1, 4, 9, 16]
```

# **12. Copying Lists and Type Functions**

### Wrong — only copies reference

```python
a = [1, 2, 3]
b = a
b.append(4)
print(a, b)
```

**Output**

```
[1, 2, 3, 4] [1, 2, 3, 4]
```

### Correct — using `.copy()`

```python
a = [1, 2, 3]
b = a.copy()
b.append(4)
print(a)
print(b)
```

**Output**

```
[1, 2, 3]
[1, 2, 3, 4]
```

### Type and isinstance

```python
x = [1, 2, 3]
print(type(x))
print(isinstance(x, list))
```

**Output**

```
<class 'list'>
True
```

### join() and list()

```python
words = ["My", "name", "is", "Majid", "Arasteh"]
print(" ".join(words))

text = "hello"
print(list(text))
```

**Output**

```
My name is Majid Arasteh
['h', 'e', 'l', 'l', 'o']
```
