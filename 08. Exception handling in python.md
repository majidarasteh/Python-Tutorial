# **Python Exception Handling — Complete Tutorial**

## **1. Introduction to Errors in Programming**

**Description:**
Errors occur when the program cannot run correctly. In Python, errors are categorized into different types depending on *when* they occur and *why* they happen.

---

## **2. Types of Errors**

### **2.1 Syntax Errors**

**Description:**
Errors caused by incorrect code structure. Python cannot interpret the code.

**Examples of causes:**

* Missing colon `:`
* Missing parentheses
* Wrong indentation

**Example:**

```python
if True
    print("Hello")
```

**Output:**

```
SyntaxError: invalid syntax
```

---

### **2.2 Logical Errors (Semantic Errors)**

**Description:**
The code runs but produces **incorrect results**.

**Example:**

```python
# Incorrect average calculation
numbers = [10, 20, 30]
avg = sum(numbers)        # Missing division by len(numbers)
print(avg)
```

**Output:**

```
60
```

(The logic is wrong; the correct answer is 20.)

---

### **2.3 Runtime Errors (Exceptions)**

**Description:**
Errors that occur **during program execution**.

**Examples:**

* Dividing by zero
* Entering letters instead of numbers
* Accessing a nonexistent file

**Example:**

```python
x = int("abc")
```

**Output:**

```
ValueError: invalid literal for int()
```

---

## **3. What is an Exception?**

**Description:**
An **exception** is a runtime error. Python provides tools to detect and handle exceptions so the program does not crash.

Example: Division by zero occurs *only* when the denominator is zero — not always — therefore it is an **exception**, not a syntax error.

---

## **4. Exception Handling Using `try` and `except`**

**Description:**
Use `try` to execute risky code, and `except` to catch and handle exceptions.

### **Methods**

* **try:** Code that might cause an exception
* **except:** Code that runs when an exception occurs

### **Example**

```python
try:
    x = float(input("Enter a number: "))
    y = float(input("Enter a number: "))
    z = x / y
    print("The result is:", z)
except:
    print("Runtime error occurred")
```

**Output (example run):**

* Input: `2` and `0`

```
Runtime error occurred
```

---

## **5. Handling Exceptions with `try`, `except`, and `else`**

**Description:**
`else` runs **only if no exception occurs**.

### **Methods**

* **else:** Runs when the try-block does *not* raise any errors

### **Example**

```python
try:
    x = float(input("Enter a number: "))
    y = float(input("Enter a number: "))
    z = x / y
except:
    print("Runtime error")
else:
    print("Result:", z)
```

**Output (normal case):**

* Input: `6` and `2`

```
Result: 3.0
```

**Output (error case):**

* Input: `5` and `0`

```
Runtime error
```

---

## **6. Full Exception Handling: `try`, `except`, `else`, `finally`**

**Description:**
`finally` always runs whether an exception occurs or not.

### **Methods**

* **finally:** Guaranteed to execute

### **Example**

```python
try:
    x = 10 / 2
except:
    print("Error!")
else:
    print("No error")
finally:
    print("End of exception block")
```

**Output:**

```
No error
End of exception block
```

---

Below are the most important Python exceptions with short explanations:

| Exception         | Description                                 |

## **8. Handling Specific Exceptions**

**Description:**
Use multiple `except` blocks to handle different exceptions individually.

### **Example**

```python
try:
    x = float(input("Enter a number: "))
    y = float(input("Enter a number: "))
    z = x / y
    print("Result:", z)

except ValueError:
    print("Value Error!")
except ZeroDivisionError:
    print("Cannot divide by zero!")
except OverflowError:
    print("Overflow Error")
except KeyboardInterrupt:
    print("Interrupted by user (Ctrl+C)")
```

**Output sample:**

* Input: `5` and `0`

```
Cannot divide by zero!
```

---

## **9. Nested Exceptions**

**Description:**
Exceptions can occur inside another exception block.

**Example:**

```python
try:
    try:
        x = int("abc")
    except ValueError:
        print("Inner ValueError")
        y = 1 / 0
except ZeroDivisionError:
    print("Outer ZeroDivisionError")
```

**Output:**

```
Inner ValueError
Outer ZeroDivisionError
```

## **10. Assertions with `assert`**

### **Description:**

`assert` checks whether a condition is True.
If False → raises `AssertionError`.

### **Method**

* **assert condition, message**

---

### **Example 1**

```python
number = float(input("Enter a positive number: "))
assert number >= 0, "The number is negative"
print(number)
```

**Output (if input = -5):**

```
AssertionError: The number is negative
```

---

### **Example 2 — Kelvin to Fahrenheit**

```python
def KelvinToFahrenheit(Temperature):
    assert Temperature >= 0, "Colder than absolute zero!"
    return ((Temperature - 273) * 1.8) + 32

print(KelvinToFahrenheit(273))
print(KelvinToFahrenheit(505.78))
print(KelvinToFahrenheit(-5))
```

**Output:**

```
32.0
451.004
Traceback (most recent call last):
AssertionError: Colder than absolute zero!
```

---

## **11. Raising Exceptions with `raise`**

### **Description:**

You can create your own exceptions using `raise`.

### **Method**

* **raise Exception("message")**

### **Example**

```python
x = 10
if x > 5:
    raise Exception("x should not exceed 5. The value was: {}".format(x))
```

**Output:**

```
Exception: x should not exceed 5. The value was: 10
```

---

## **12. Warnings**

**Description:**
Warnings alert the programmer about unexpected behavior but do *not* stop the program. You can generate or control warnings using the `warnings` module.

**Example:**

```python
import warnings

age = -5

if age < 0:
    warnings.warn("Age cannot be negative! Using default value 0.")

print("Age is:", max(age, 0))

```

**Output**

```
Age is: 0
C:\Users\Majid\AppData\Local\Temp\ipykernel_5300\3131632650.py:6: UserWarning: Age cannot be negative! Using default value 0.
  warnings.warn("Age cannot be negative! Using default value 0.")
```

## **13. Summary Table of Exception Handling Commands**

| Method / Keyword    | Short Description                                         | Example                           |
| ------------------- | --------------------------------------------------------- | --------------------------------- |
| **try**             | Block of risky code                                       | `try: x = 1/0`                    |
| **except**          | Handles the exception                                     | `except ZeroDivisionError:`       |
| **else**            | Runs if no error occurs                                   | `else: print("OK")`               |
| **finally**         | Always runs                                               | `finally: print("Done")`          |
| **raise**           | Creates a custom exception                                | `raise Exception("Error")`        |
| **assert**          | Validates a condition; raises AssertionError              | `assert x >= 0, "Negative!"`      |
| **warnings.warn()** | Generates a non-fatal warning message (program continues) | `warnings.warn("Invalid value!")` |
