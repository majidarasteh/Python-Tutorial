# Numerical Data Types
In Python, **numerical data types** are used to **store numbers**. Python has three main numerical types:
1. **Integer (`int`)**
   * Stores whole numbers (positive, negative, or zero).
   * Examples:
  
     ```python
        x = 10
        y = -5
        z = 0
     ```
2. **Float (`float`)**
   * Stores **decimal numbers** (numbers with a fractional part).
   * Examples:
  
     ```python
        pi = 3.14
        temperature = -0.5
        height = 2.0
     ```

    * Scientific Notation
      * e or E means "√ó 10 to the power of".
      * $6e8 ‚Üí 6√ó10^8 ‚Üí 600000000.0$
      * $6e-8 ‚Üí 6√ó10^{-8} ‚Üí 0.00000006$

  3. **Complex (`complex`)**
     * Stores complex numbers with a **real part** and an **imaginary part**.
     * Imaginary part is denoted with a  `j`.
     * Examples:

       ```python
          a = 2 + 3j
          b = 1 - 4j
       ```

**Note: Use int for counting, float for measurements, and complex for mathematical computations involving complex numbers.**

## Decimal Numbers in Python
* Decimal numbers are numbers that have a **fractional part**, i.e., numbers with digits after the decimal point.
* In Python, they are usually **represented** by the `float` data type.
* Example of decimal numbers:

  ```python
    x = 3.14
    y = -0.5
    z = 2.0
  ```
  
**Important Points**
1. **Float Type**
   * Decimal numbers in Python are `float` by default.
   * They can be positive or negative.
   * Even a whole number with a decimal point (e.g., `2.0`) is **considered a float**.
     
2. **Scientific Notation**: You can also write decimal numbers in scientific notation:
   
   ```python
      a = 6e-8   # 6 √ó 10^-8 ‚Üí 0.00000006
      b = 1.23e4 # 1.23 √ó 10^4 ‚Üí 12300.0
   ```

3. **Precision**
   * Float numbers have **limited precision**.
   * For very **precise decimal calculations** (like financial calculations), you can use the **decimal module**:
  
     ```python
        from decimal import Decimal

        price = Decimal('19.99')
        tax = Decimal('0.07')
        total = price + price * tax
        print(total)  # 21.3893
     ```

      | Type      | Example          | Description                     |
      | --------- | ---------------- | ------------------------------- |
      | `float`   | 3.14, -0.5, 2.0  | Decimal numbers (with fraction) |
      | `Decimal` | Decimal('19.99') | High-precision decimal numbers  |

    * Due to the way floating-point numbers are represented in binary within computer systems, decimal values like 0.1 and 0.2 cannot be stored with perfect accuracy. When you define these numbers as floats in Python, they are often stored as approximations, such as 0.10000000000000000555 or 0.20000000000000001110. This inherent imprecision can lead to unexpected results in calculations‚Äîfor instance, (0.1 + 0.2 == 0.3) surprisingly evaluates to False. This is a common source of logical errors in programs that require high precision for financial or scientific computations.
   
## Fraction Data Type
* Python has a Fraction data type in the fractions module to represent rational numbers (numbers as a ratio of two integers).
* A fraction is represented as **numerator/denominator**.
* It allows exact arithmetic with **fractions instead of floating-point approximations**.

  ```python
    from fractions import Fraction
    
    # Create fractions
    a = Fraction(3, 4)    # 3/4
    b = Fraction(5, 6)    # 5/6
    
    # Arithmetic with fractions
    print(a + b)           # 19/12
    print(a - b)           # -1/12
    print(a * b)           # 5/8
    print(a / b)           # 9/10
    
    # Fraction from float or string
    c = Fraction(0.75)     # 3/4
    d = Fraction("0.6")    # 3/5
  ```

* Fractions maintain exact values:

  ```python
     Fraction(1, 3) + Fraction(1, 3)  # 2/3
  ```
  
* Fractions can be converted to float if needed:

  ```python
     float(Fraction(3, 4))  # 0.75
  ```
  
* Useful for precise arithmetic, like in mathematics, engineering, or finance, where floating-point rounding errors are undesirable.

## Binary, Octal, and Hexadecimal Numbers in Python
1. **Binary Numbers**
   * Binary numbers use **base 2**, consisting of digits `0` and `1`.
   * In Python, **binary literals start with** `0b` or `0B`.

     ```python
        x = 0b1010  # binary for 10
        print(x)    # Output: 10
     ```
2. **Octal Numbers**:
   * Octal numbers use **base 8**, consisting of digits `0` to `7`.
   * In Python, octal **literals start with** `0o` or `0O`.
  
     ```python
        y = 0o12   # octal for 10
        print(y)   # Output: 10
     ```
3. **Hexadecimal Numbers**:
   * Hexadecimal numbers use **base 16**, consisting of **digits** `0-9` and **letters** `A-F` **(or a-f) for values 10-15**.
   * In Python, **hexadecimal literals start with** `0x` or `0X`.

     ```python
        z = 0xA    # hexadecimal for 10
        print(z)   # Output: 10
     ```

    | Number System | Base | Digits   | Python Prefix | Example  | Decimal Value |
    | ------------- | ---- | -------- | ------------- | -------- | ------------- |
    | Binary        | 2    | 0,1      | `0b` / `0B`   | `0b1010` | 10            |
    | Octal         | 8    | 0-7      | `0o` / `0O`   | `0o12`   | 10            |
    | Hexadecimal   | 16   | 0-9, A-F | `0x` / `0X`   | `0xA`    | 10            |

**You can convert between these formats and decimal using Python built-in functions:**

```python
bin(10)   # '0b1010'
oct(10)   # '0o12'
hex(10)   # '0xa'
```

# Strings in Python
* A string is a **sequence of characters** enclosed in single quotes `'...'` or double quotes `"..."`.
* Strings can contain **letters, numbers, symbols, or even spaces**.

  ```python
  name = "Ali"
  password = 'jf38jkwld213'
  ```

* **Docstrings**: A docstring (documentation string) is a special type of string used to document functions, classes, and modules in Python.
Docstrings are written using three double quotes:

  ```python
      string = """Hello,
      I
      Hope
      you are
      doinq well. """
      print(string)
  ```

* **Example: Docstring in a Function**

  ```python
    def add(a, b):
        """
        This function takes two numbers (a and b)
        and returns their sum.
        """
        return a + b
  ```

* When a string's content needs to include a quotation mark or an apostrophe, it must be preceded by a backslash (`\`). This is known as escaping the character. For example: name = `"I\'m Majid"`.
* In Python, **unlike** many other programming languages, **there is no distinct 'character' data type**. A single character is simply treated as a string with a length of one.

### Encoding
* **Converts characters into numbers** so that computers can store or transmit text.
* Example: **'A' ‚Üí 65 in ASCII**
* In Python: `ord()` **gives the numeric code of a character**

```python
   print(ord('A'))  # Output: 65
```

### Decoding
* **Converts numbers back into characters**.
* In Python: `chr()` gives the character from its numeric code

```python
   print(chr(65))   # Output: 'A'
```

### Character Encoding Standards
1. **ASCII (American Standard Code for Information Interchange)**
   * **Uses 7 bits** to represent **128 characters**.
   * Includes English letters, digits, punctuation, and control characters.
   * **Examples**:
     * 'A' ‚Üí 65
     * 'a' ‚Üí 97
     * '0' ‚Üí 48
       
2. **Unicode (UNI code)**
   * Can **represent all characters from all languages**.
   * Python strings are Unicode by default.
   * **Unicode covers ASCII** and also characters like ÿ®, ‰Ω†, üòä.
  
     ```python
        print(ord('ÿ®'))  # 1576
        print(ord('üòä')) # 128522
     ```

| Function / Concept | Description                      | Example       | Output      |
| ------------------ | -------------------------------- | ------------- | ----------- |
| `ord()`            | Character ‚Üí ASCII / Unicode code | `ord('A')`    | 65          |
| `chr()`            | ASCII / Unicode code ‚Üí Character | `chr(65)`     | 'A'         |
| ASCII              | Standard 128 characters          | 'A', 'a', '0' | 65,97,48    |
| Unicode            | Supports all characters globally | '‡§Ö', 'üòä'     | 2309,128522 |


## Accessing Characters of a String in Python
* A string is a sequence of characters, and each character has an index (position).
* You can access characters using square brackets `[ ]`.

  ```python
    Index:   0   1   2   3   4
    String:  P   y   t   h   o   n
    Index:  -6  -5  -4  -3  -2  -1
  ```

* Example

  ```python
    s = "Python"
    
    print(s[0])   # P
    print(s[1])   # y
    print(s[5])   # n
  ```

* **Negative Indexing**

  ```python
    s = "Python"
    
    print(s[-1])   # n (last character)
    print(s[-2])   # o
    print(s[-6])   # P
  ```

* **Slicing Strings**: You can extract a substring using slicing:

  ```python
     string[start : end]
  ```
  * start = included
  * end = excluded

* **Example**

  ```python
     s = "Python"
     
     print(s[0:2])   # Py
     print(s[2:5])   # tho
     print(s[:4])    # Pyth
     print(s[3:])    # hon
     print(s[:])     # Python (whole string)
  ```

* **Slicing with Step**

  ```python
     string[start : end : step]
  ```

  **Example**

  ```python
    print(s[::2])    # Pto  (every second character)
    print(s[::-1])   # nohtyP (reversed string)
  ```

* **Strings Are Immutable**: You can access characters but cannot change them:

  ```python
     s[0] = "A"   # ‚ùå Error
  ```
  
  **To modify a string, you must create a new one:**
  
  ```python
     s = "Python"
     s = "A" + s[1:]   # "Aython"
  ```
* Indices start at `0`.
* Attempting to access a character **outside the index** range will result in an `IndexError`.
* The **index must be an integer**, using a non-integer value will cause a `TypeError`.
* You **cannot delete individual characters** from a string. However, it is **possible to delete the entire string** using the `del` keyword.
  
  ```python
     Str = "Hello"
     del Str
  ```

## Concatenation of Strings (`+`) and Repetition of Strings (`*`) in Python
1. **String Concatenation ‚Äî Using +**: Concatenation means joining two or more strings together. Both operands must be strings (you cannot add a string to an integer directly)

   ```python
      s1 = "Hello"
      s2 = "World"
      result = s1 + " " + s2
      print(result)            # Hello World
   ```
   
2. **String Repetition ‚Äî Using `*`**: The `*` operator repeats a string multiple times.

   ```python
      text = "Hi "
      print(text * 3)   # Hi Hi Hi 
   ```

## 'Checking Substrings in Python Using `in`
* Using the `in` keyword, you can check whether a substring exists within a string or not.
* It **returns a Boolean** value (`True` or `False`).
* The `in` operator is **case-sensitive**.

  ```python
     text = "Python programming"
     print("Python" in text)      # True

     print("Java" in text)        # False

     print("C++" not in text)     # True 
  ```

# Some Python String Built-in Methods

### `count()`
Returns how many times a substring appears in the string.

```python
   s = "banana"
   print(s.count("a"))   # 3
```

### `enumerate()`
Returns index + character when looping through a string.

```python
   s = "cat"
   for index, char in enumerate(s):
       print(index, char)
```

### `len()`
Returns the length of the string.

```python
   len("hello")  # 5
```

### `upper()`
Converts all characters to uppercase.

```python
   "python".upper()   # "PYTHON"
```

### `lower()`
Converts all characters to lowercase.

```python
   "PyThOn".lower()  # "python"
```

### `split()`
Splits the string into a list based on a separator.

```python
   "one two three".split()  
   # ["one", "two", "three"]
```

### `join()`
Joins elements of a list into a single string.

```python
   "-".join(["a", "b", "c"])  
   # "a-b-c"
```

### `find()`
* Returns the index of the first occurrence of a substring.
* Returns -1 if not found.

```python
   "hello".find("l")   # 2
```

### `replace()`
Replaces parts of the string.

```python
   "hello world".replace("world", "Python")
   # "hello Python"
```

### `str()`
Converts numbers or objects to strings.

```python
   str(123)  # "123"
```

### `capitalize()`
capitalize()

```python
   "python".capitalize()  
   # "Python"
```

### `startswith()`
Checks if a string starts with a specific substring.

```python
   "hello".startswith("he")   # True
```

### `endswith()`
Checks if a string ends with a specific substring.

```python
   "hello".endswith("lo")   # True
```

### `isalnum()`
Returns True if the string has only letters and numbers.

```python
   "abc123".isalnum()   # True
```

### `isdigit()`
Returns True if the string contains only digits.

```python
   "12345".isdigit()  # True
```

### `isidentifier()`
Checks if the string is a valid Python variable name.

```python
   "age1".isidentifier()  # True
   "1age".isidentifier()  # False
```

### `isnumeric()`
True if the string has numeric characters (digits, fractions, Unicode numbers).

```python
   "¬Ω".isnumeric()   # True
```

### `isprintable()`
True if all characters are printable.

```python
   "Hello\nWorld!".isprintable()  # False
   "Hello World!".isprintable()  # True
```

### `isspace()`
True if the string has only whitespace characters.

```python
   "   ".isspace()     # True
   "  S  ".isspace()   # False
```

### `istitle()`
True if each word starts with a capital letter.

```python
   "Hello World".istitle()  # True
```

### `partition()`
Splits the string into 3 parts: (before separator, separator, after separator)

```python
  txt = "I could eat bananas all day"
  print(txt.partition("bananas"))
  # ('I could eat ', 'bananas', ' all day')
```

### `strip()`
Removes leading and trailing whitespace.

```python
   "  hello  ".strip()
   # "hello"
```

### `swapcase()`
Changes uppercase to lowercase and vice versa.

```python
   "PyThOn".swapcase()
   # "pYtHoN"
```
