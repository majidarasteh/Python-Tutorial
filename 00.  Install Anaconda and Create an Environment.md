#  Install Anaconda and Create an Environment
1. Step 1: Install Anaconda
   * Go to https://www.anaconda.com/download
   * Download and install the latest Python version for your operating system.
   * Follow the setup wizard (use default options if unsure).
     
2. Step 2: Create & Setup Your AI Environment
   * Open Anaconda Prompt (Windows) or your Terminal (macOS/Linux) and run these commands:
     
     ```
        # 1. Create a new environment named 'AI_env' with Python e.g. 3.10
          conda create --name AI_env python=3.10 -y
          
          # 2. Activate the new environment
          conda activate AI_env
          
          # 3. Install essential AI and data science libraries
          conda install -y numpy pandas matplotlib scikit-learn pytorch torchvision torchaudio tensorflow
     ```

     Your environment is now ready for development.

3. Step 3: (Optional) How to Remove the Environment
   * If you need to delete the environment later:

     ```
        # 1. First, deactivate the current environment
        conda deactivate

        # 2. Remove the 'AI_env' environment and all its packages
        conda remove --name AI_env --all
        
        # 3. Clear package cache to free up disk space
        conda clean --all -y
        
        # 4. Update Conda itself
        conda update -n base conda -y
     ```

# Launching Your Environment & Using Jupyter

1. **Open Anaconda Navigator from your Start Menu or Applications.**
2. **Select Your Environment:**
   * On the left sidebar, click on "**Environments"**.
   * From the dropdown list, select your created environment: `AI_env`.
3. **Launch Jupyter Lab / Notebook:**
   * In the main panel, find and click the "**Install**" button under Jupyter Lab or Jupyter Notebook.
   * Once installed, the button will change to "**Launch**". Click it to open your IDE.

4. **Start Coding:**
   * Create a new notebook (`File` > `New` > `Notebook`).
   * You can now write your codes in your code cells.
  
## Quick Command Line Method:
Alternatively, from Anaconda Prompt:

```
    conda activate AI_env
    jupyter lab
```
