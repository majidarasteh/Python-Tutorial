# Python Set Data Type 

## 1. Introduction to Sets

A set is an unordered collection of unique elements in Python. Sets cannot contain duplicates, and **elements must be immutable (like numbers, strings, or tuples)**. The **set itself is mutable**, meaning you can add or remove elements.

---

## 2. Creating Sets

Sets are created using curly braces `{}` or the `set()` function.

**Examples:**

```python
my_set = {10, 20, 30, 20, 10}
print(my_set)
# Output: {10, 20, 30}  (duplicates removed)

my_set = {"Python", 3.14, (1, 2)}
print(my_set)
# Output: {3.14, 'Python', (1, 2)}

my_set = set([5, 10, 5, 15])
print(my_set)
# Output: {5, 10, 15}
```

---

## 3. Creating Empty Sets

Using `{}` creates an empty dictionary, not a set. To create an empty set, use `set()`.

**Examples:**

```python
a = {}
print(type(a))
# Output: <class 'dict'>

a = set()
print(type(a))
# Output: <class 'set'>
```

---

## 4. Adding Elements to Sets

* `add(element)`: Adds a single element.
* `update(iterable)`: Adds multiple elements from list, tuple, string, or another set.

**Examples:**

```python
my_set = {10, 30}

my_set.add(20)
print(my_set)
# Output: {10, 20, 30}

my_set.update([20, 40])
print(my_set)
# Output: {10, 20, 30, 40}

my_set.update([50, 60], {10, 70})
print(my_set)
# Output: {70, 40, 10, 50, 20, 60, 30}
```

---

## 5. Removing Elements from Sets (`discard` vs `remove`)

* `discard(element)`: Removes element if it exists; no error if it doesn't.
* `remove(element)`: Removes element if it exists; raises `KeyError` if it doesn't.

**Examples:**

```python
my_set = {10, 20, 30, 40, 50}

my_set.discard(30)
print(my_set)
# Output: {10, 20, 40, 50}

my_set.remove(50)
print(my_set)
# Output: {10, 20, 40}

my_set.discard(100)
print(my_set)
# Output: {10, 20, 40}  (no change)

# my_set.remove(100)  # Uncommenting will raise KeyError: 100
```

---

## 6. Removing Elements from Sets (`pop` and `clear`)

* `pop()`: Removes and returns a random element.
* `clear()`: Removes all elements.

**Examples:**

```python
my_set = {10, 20, 30, 40}

print(my_set.pop())  # Output: random element
my_set.clear()
print(my_set)  # Output: set()
```

---

## 7. Set Operations: Union

Union returns all elements from both sets. Use `|` operator or `union()` method.

**Examples:**

```python
A = {1, 2, 3, 4}
B = {3, 4, 5, 6}

print(A | B)
# Output: {1, 2, 3, 4, 5, 6}

print(A.union(B))
# Output: {1, 2, 3, 4, 5, 6}
```

---

## 8. Set Operations: Intersection

Intersection returns elements common to both sets. Use `&` operator or `intersection()` method.

**Examples:**

```python
A = {1, 2, 3, 4}
B = {3, 4, 5, 6}

print(A & B)
# Output: {3, 4}

print(A.intersection(B))
# Output: {3, 4}
```

---

## 9. Set Operations: Difference

Difference returns elements in one set but not the other. Use `-` operator or `difference()` method.

**Examples:**

```python
A = {1, 2, 3, 4}
B = {3, 4, 5, 6}

print(A - B)
# Output: {1, 2}

print(B.difference(A))
# Output: {5, 6}
```

---

## 10. Set Operations: Symmetric Difference

Returns elements in either set but not both. Use `^` operator or `symmetric_difference()` method.

**Examples:**

```python
A = {1, 2, 3, 4}
B = {3, 4, 5, 6}

print(A ^ B)
# Output: {1, 2, 5, 6}

print(A.symmetric_difference(B))
# Output: {1, 2, 5, 6}
```

---

## 11. Checking Element Membership

* `in`: Checks if an element exists.
* `not in`: Checks if an element does not exist.

**Examples:**

```python
test_set = {10, 20, 30, 40}

print(25 in test_set)
# Output: False

names_set = {"Ali", "Sara", "Nima", "Lina"}
print("Mina" not in names_set)
# Output: True
```

---

## 12. Checking Set Equality

Two sets are equal if they contain the same elements, regardless of order.

**Examples:**

```python
set1 = {'apple', 'orange'}
set2 = {'orange', 'apple'}
print(set1 == set2)
# Output: True
```

---

## 13. Other Set Methods

* `copy()`: Returns a shallow copy of the set.
* `len()`: Returns the number of elements.
* `isdisjoint()`: True if no common elements.
* `issubset()`: True if all elements are in another set.
* `issuperset()`: True if it contains all elements of another set.
* `difference_update()`: Removes all elements of another set from this set.
* `symmetric_difference_update()`: Keeps only unique elements in both sets.

---

## 14. Frozenset

A frozenset is immutable. Methods that add/remove elements are not available. Frozensets are hashable and can be used as dictionary keys.

**Examples:**

```python
A = frozenset([1, 2, 3, 4])
B = frozenset([3, 4, 5, 6])

print(A.union(B))
# Output: frozenset({1, 2, 3, 4, 5, 6})
```
