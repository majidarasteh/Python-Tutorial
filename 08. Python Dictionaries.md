# Python Dictionary

Dictionaries in Python are powerful and flexible data structures used to store information as key–value pairs. Unlike lists, which are indexed by position, dictionaries allow you to access data using meaningful keys such as strings or numbers. They are unordered, mutable, and can store any data type, including other dictionaries—making them ideal for representing structured information like records, configurations, and JSON-like data. Python provides a rich set of methods for adding, removing, updating, and searching dictionary items, which makes them one of the most essential and versatile tools in the language.


---

### **1. Dictionary in Python**

**Description:**
A dictionary maps a key to a value. It can be considered an "associative array". Dictionaries are similar to HashMaps in Java and can store any type of data as values.

**Example:**

```python
phone_book = {'John': '123-456', 'Alice': '987-654', 'Bob': '555-555'}
print(phone_book)
```

**Output:**

```
{'John': '123-456', 'Alice': '987-654', 'Bob': '555-555'}
```

---

### **2. Dictionary Characteristics**

**Description:**
Dictionaries are mutable, keys must be unique, and values can be duplicated. Keys can be integers, floats, strings, or tuples, but not lists or other dictionaries. Nested dictionaries are allowed.

**Example:**

```python
d = {(1, 1): 'a', (1, 2): 'b', (2, 1): 'c', (2, 2): 'd'}
print(d[(1,1)])
print(d[(2,1)])
```

**Output:**

```
a
c
```

---

### **3. How to Create a Dictionary**

**Description:**
Dictionaries can be created using `{}`, `dict()`, `dict()` with `zip()`, `dict.fromkeys()`, or dictionary comprehension.

#### **3.1 Using Curly Braces**

```python
MLB_team = {
    'Colorado': 'Rockies',
    'Boston': 'Red Sox',
    'Minnesota': 'Twins'
}
print(MLB_team)
```

**Output:**

```
{'Colorado': 'Rockies', 'Boston': 'Red Sox', 'Minnesota': 'Twins'}
```

#### **3.2 Using dict() Function**

```python
MLB_team = dict([
    ('Colorado', 'Rockies'),
    ('Boston', 'Red Sox'),
    ('Minnesota', 'Twins')
])
print(MLB_team)
```

**Output:**

```
{'Colorado': 'Rockies', 'Boston': 'Red Sox', 'Minnesota': 'Twins'}
```

#### **3.3 Using dict() with Named Parameters**

```python
MLB_team = dict(
    Colorado='Rockies',
    Boston='Red Sox',
    Minnesota='Twins'
)
print(MLB_team)
```

**Output:**

```
{'Colorado': 'Rockies', 'Boston': 'Red Sox', 'Minnesota': 'Twins'}
```

#### **3.4 Using dict() with zip()**

```python
keys = ['a', 'b', 'c']
values = [1, 2, 3]
myDict = dict(zip(keys, values))
print(myDict)
```

**Output:**

```
{'a': 1, 'b': 2, 'c': 3}
```

#### **3.5 Dictionary Comprehension**

```python
keys = ['a', 'b', 'c']
values = [1, 2, 3]
myDict = {k: v for (k, v) in zip(keys, values)}
print(myDict)
```

**Output:**

```
{'a': 1, 'b': 2, 'c': 3}
```

#### **3.6 Using dict.fromkeys()**

```python
dic = dict.fromkeys(range(5), True)
print(dic)
```

**Output:**

```
{0: True, 1: True, 2: True, 3: True, 4: True}
```

### **4. Accessing Dictionary Values**

**Description:**
Values can be accessed using keys or the `get()` method.

#### **4.1 Using Keys**

```python
dic1 = {'tel1':'09011234567', 'tel2':'08301234567'}
print(dic1['tel1'])
```

**Output:**

```
09011234567
```

#### **4.2 Using get()**

```python
d = {'a': 10, 'b': 20, 'c': 30}
print(d.get('b'))
print(d.get('z'))
print(d.get('z', -1))
```

**Output:**

```
20
None
-1
```

#### **4.3 Using setdefault()**

```python
dict1 = {'Name': 'Zara', 'Age': 7}
print(dict1.setdefault('Age', None))
print(dict1.setdefault('Sex', None))
```

**Output:**

```
7
None
```

### **5. Editing a Dictionary**

**Description:**
Assign a new value to an existing key to update the dictionary.

```python
MLB_team = {'Colorado': 'Rockies', 'Boston': 'Red Sox', 'Minnesota': 'Twins'}
MLB_team['Minnesota'] = 'Vikings'
print(MLB_team)
```

**Output:**

```
{'Colorado': 'Rockies', 'Boston': 'Red Sox', 'Minnesota': 'Vikings'}
```

---

### **6. Deleting Values from a Dictionary**

**Description:**
Use `del` to remove a key-value pair.

```python
MLB_team = {'Colorado': 'Rockies', 'Boston': 'Red Sox', 'Minnesota': 'Twins'}
del MLB_team['Minnesota']
print(MLB_team)
```

**Output:**

```
{'Colorado': 'Rockies', 'Boston': 'Red Sox'}
```

### **7. Dictionary Methods Overview**

**Description:**
Common methods include `len()`, `clear()`, `items()`, `keys()`, `values()`, `pop()`, `popitem()`, `update()`, and `copy()`.

#### **7.1 len()**

```python
d = {'Name': 'Zara', 'Age': 7}
print(len(d))
```

**Output:**

```
2
```

#### **7.2 clear()**

```python
d = {'a': 10, 'b': 20}
d.clear()
print(d)
```

**Output:**

```
{}
```

#### **7.3 items()**

```python
d = {'a': 10, 'b': 20}
print(list(d.items()))
```

**Output:**

```
[('a', 10), ('b', 20)]
```

#### **7.4 keys() and values()**

```python
d = {'a': 10, 'b': 20}
print(list(d.keys()))
print(list(d.values()))
```

**Output:**

```
['a', 'b']
[10, 20]
```

#### **7.5 pop()**

```python
d = {'a': 10, 'b': 20}
print(d.pop('b'))
print(d.pop('z', -1))
print(d)
```

**Output:**

```
20
-1
{'a': 10}
```

#### **7.6 popitem()**

```python
d = {'a': 10, 'b': 20, 'c': 30}
print(d.popitem())
print(d)
```

**Output:**

```
('c', 30)
{'a': 10, 'b': 20}
```

#### **7.7 update()**

```python
d1 = {'a': 10, 'b': 20}
d2 = {'b': 200, 'c': 300}
d1.update(d2)
print(d1)
```

**Output:**

```
{'a': 10, 'b': 200, 'c': 300}
```

#### **7.8 copy()**

```python
original = {1: 'geeks', 2: 'for'}
new = original.copy()
new.clear()
print('new:', new)
print('original:', original)
```

**Output:**

```
new: {}
original: {1: 'geeks', 2: 'for'}
```

### **8. Nested Dictionaries**

**Description:**
Dictionaries can contain other dictionaries as values.

**Example 1: Create nested dictionaries**

```python
people = {
    1: {'name': 'John', 'age': '27', 'sex': 'Male'},
    2: {'name': 'Marie', 'age': '22', 'sex': 'Female'}
}
print(people)
```

**Output:**

```
{1: {'name': 'John', 'age': '27', 'sex': 'Male'}, 
 2: {'name': 'Marie', 'age': '22', 'sex': 'Female'}}
```

**Example 2: Accessing nested dictionary**

```python
print(people[1]['name'])
print(people[1]['age'])
print(people[1]['sex'])
```

**Output:**

```
John
27
Male
```

**Example 3: Adding nested dictionary items**

```python
people[3] = {}
people[3]['name'] = 'Luna'
people[3]['age'] = '24'
people[3]['sex'] = 'Female'
people[3]['married'] = 'No'
print(people[3])
```

**Output:**

```
{'name': 'Luna', 'age': '24', 'sex': 'Female', 'married': 'No'}
```

**Example 4: Deleting specific nested keys**

```python
del people[3]['married']
print(people[3])
```

**Output:**

```
{'name': 'Luna', 'age': '24', 'sex': 'Female'}
```

**Example 5: Deleting multiple nested items**

```python
del people[3]
print(people)
```

**Output:**

```
{1: {'name': 'John', 'age': '27', 'sex': 'Male'}, 
 2: {'name': 'Marie', 'age': '22', 'sex': 'Female'}}
```

### **9. Checking Key Existence**

**Description:**
Use the `in` keyword to check if a key exists in a dictionary.

```python
a = {'name': 'John', 'age': 21}
print('name' in a)
print('address' in a)
```

**Output:**

```
True
False
```

### **10. Iterating Through Dictionaries**

**Description:**
Use `for` loops or the `items()` method to iterate over dictionaries.

```python
num1 = {0: '00', 1: '01', 2: '10', 3: '11'}
for k in num1:
    print(k, ":", num1[k])
```

**Output:**

```
0 : 00
1 : 01
2 : 10
3 : 11
```

```python
Ages = {'ali': 12, 'mohammad': 20, 'Hamed': 27, 'zahra': 25}
for k, v in Ages.items():
    print(k, v)
```

**Output:**

```
ali 12
mohammad 20
Hamed 27
zahra 25
```

```python
a = {'name': 'amir', 'age': 10}
for item in a.items():
    print(item)
```

**Output:**

```
('name', 'amir')
('age', 10)
```

### **11. Comparing Dictionaries (cmp Method)**

**Description:**
The `cmp()` method compares two dictionaries (Python 2 only). Returns `0` if equal, `1` if first is greater, `-1` if first is smaller.

```python
dict1 = {'Name': 'Zara', 'Age': 7}
dict2 = {'Name': 'Mahnaz', 'Age': 27}
dict4 = {'Name': 'Zara', 'Age': 7}

# Python 2 example:
# cmp(dict1, dict2)  # Output: -1
# cmp(dict1, dict4)  # Output: 0

# Python 3 alternative:
print(dict1 == dict2)  # False
print(dict1 == dict4)  # True
```

**Output (Python 3):**

```
False
True
```
