# Python Dictionary

**Dictionaries** in Python are powerful and flexible data structures used to **store information as key–value pairs**. Unlike lists, which are indexed by position, dictionaries allow you to access data using meaningful keys such as strings or numbers. They are **unordered, mutable, and can store any data type, including other dictionaries**, making them ideal for representing structured information like records, configurations, and JSON-like data. Python provides a rich set of methods for **adding, removing, updating, and searching dictionary items**, which makes them one of the most essential and versatile tools in the language.

---

### **1. Dictionary in Python**

**Description:**
A dictionary maps a key to a value. It can be considered an "associative array". Dictionaries are similar to HashMaps in Java and can store any type of data as values.

**Example:**

```python
phone_book = {'John': '123-456', 'Alice': '987-654', 'Bob': '555-555'}
print(phone_book)
```

**Output:**

```
{'John': '123-456', 'Alice': '987-654', 'Bob': '555-555'}
```

---

### **2. Dictionary Characteristics**

**Description:**
Dictionaries are mutable, keys must be unique, and values can be duplicated. Keys **can be integers, floats, strings, or tuples, but not lists or other dictionaries**. Nested dictionaries are allowed.

**Example:**

```python
d = {(1, 1): 'a', (1, 2): 'b', (2, 1): 'c', (2, 2): 'd'}
print(d[(1,1)])
print(d[(2,1)])
```

**Output:**

```
a
c
```

### **3. How to Create a Dictionary**

**Description:**
Dictionaries can be created using `{}`, `dict()`, `dict()` with `zip()`, `dict.fromkeys()`, or dictionary comprehension.

#### **3.1 Using Curly Braces**

```python
MLB_team = {
    'Colorado': 'Rockies',
    'Boston': 'Red Sox',
    'Minnesota': 'Twins'
}
print(MLB_team)
```

**Output:**

```
{'Colorado': 'Rockies', 'Boston': 'Red Sox', 'Minnesota': 'Twins'}
```

#### **3.2 Using dict() Function**

```python
MLB_team = dict([
    ('Colorado', 'Rockies'),
    ('Boston', 'Red Sox'),
    ('Minnesota', 'Twins')
])
print(MLB_team)
```

**Output:**

```
{'Colorado': 'Rockies', 'Boston': 'Red Sox', 'Minnesota': 'Twins'}
```

#### **3.3 Using dict() with Named Parameters**

```python
MLB_team = dict(
    Colorado='Rockies',
    Boston='Red Sox',
    Minnesota='Twins'
)
print(MLB_team)
```

**Output:**

```
{'Colorado': 'Rockies', 'Boston': 'Red Sox', 'Minnesota': 'Twins'}
```

#### **3.4 Using dict() with zip()**

Using `zip()` inside a dictionary comprehension allows you to combine two iterables, one containing keys and the other containing values, into a dictionary in a clean and efficient way. The `zip()` function pairs elements by their position, and the comprehension builds key–value pairs from those matched items, making it ideal for converting two lists into a single dictionary.

```python
keys = ['a', 'b', 'c']
values = [1, 2, 3]
myDict = dict(zip(keys, values))
print(myDict)
```

**Output:**

```
{'a': 1, 'b': 2, 'c': 3}
```

#### **3.5 Using dict.fromkeys()**

The `dict.fromkeys()` method creates a new dictionary by using the elements of an iterable as keys and **assigning all of them the same value**. It is a quick way to **initialize multiple keys with a default value**, such as `None` or any value you choose.

```python
dic = dict.fromkeys(range(5), True)
print(dic)
```

**Output:**

```
{0: True, 1: True, 2: True, 3: True, 4: True}
```

### **4. Accessing Dictionary Values**

**Description:**
Values can be accessed using keys or the `get()` method.

#### **4.1 Using Keys**

```python
dic1 = {'tel1':'09011234567', 'tel2':'08301234567'}
print(dic1['tel1'])
```

**Output:**

```
09011234567
```

#### **4.2 Using get()**

```python
d = {'a': 10, 'b': 20, 'c': 30}
print(d.get('b'))
print(d.get('z'))
print(d.get('z', -1))
```

**Output:**

```
20
None
-1
```
When using `d.get('z', -1)`, the value `-1` is a default value that will be returned if the **key 'z' does not exist in the dictionary**. **Instead of raising** an `error`, `get()` safely **returns the default you provide**. In this case, `-1` simply acts as a placeholder **meaning “not found.”**

#### **4.3 Using setdefault()**

The `setdefault()` method **returns the value of a key if it already exists** in the dictionary; **otherwise, it adds the key with a specified default value** and then returns that value. 

```python
dict1 = {'Name': 'Zara', 'Age': 7}
print(dict1.setdefault('Age', None))
print(dict1.setdefault('Sex', None))
```

**Output:**

```
7
None
```

### **5. Editing a Dictionary**

**Description:**
Assign a new value to an existing key to update the dictionary.

```python
MLB_team = {'Colorado': 'Rockies', 'Boston': 'Red Sox', 'Minnesota': 'Twins'}
MLB_team['Minnesota'] = 'Vikings'
print(MLB_team)
```

**Output:**

```
{'Colorado': 'Rockies', 'Boston': 'Red Sox', 'Minnesota': 'Vikings'}
```

---

### **6. Deleting Values from a Dictionary**

**Description:**
Use `del` to remove a key-value pair.

```python
MLB_team = {'Colorado': 'Rockies', 'Boston': 'Red Sox', 'Minnesota': 'Twins'}
del MLB_team['Minnesota']
print(MLB_team)
```

**Output:**


### **7. Dictionary Methods Overview**

**Description:**
Common methods include `len()`, `clear()`, `items()`, `keys()`, `values()`, `pop()`, `popitem()`, `update()`, and `copy()`.

#### **7.1 len()**

Returns the **number of key–value pairs** in the dictionary.

```python
d = {'Name': 'Zara', 'Age': 7}
print(len(d))
```

**Output:**

```
2
```

#### **7.2 clear()**

**Removes all items** from the dictionary, leaving it **empty**.

```python
d = {'a': 10, 'b': 20}
d.clear()
print(d)
```

**Output:**

```
{}
```

#### **7.3 items()**

Returns a view object containing **all dictionary key–value pairs as tuples**.

```python
d = {'a': 10, 'b': 20}
print(list(d.items()))
```

**Output:**

```
[('a', 10), ('b', 20)]
```

#### **7.4 keys() and values()**

* **`keys()`** — Returns a view object containing all the keys in the dictionary.
* **`values()`** — Returns a view object containing all the values in the dictionary.

```python
d = {'a': 10, 'b': 20}
print(list(d.keys()))
print(list(d.values()))
```

**Output:**

```
['a', 'b']
[10, 20]
```

#### **7.5 pop()**

**Removes and returns the value of a specified key**; raises an error if the key does not exist (unless a default is provided).

```python
d = {'a': 10, 'b': 20}
print(d.pop('b'))
print(d.pop('z', -1))
print(d)
```

**Output:**

```
20
-1
{'a': 10}
```

#### **7.6 popitem()**

Removes and returns the **last inserted key–value pair** as a tuple.

```python
d = {'a': 10, 'b': 20, 'c': 30}
print(d.popitem())
print(d)
```

**Output:**

```
('c', 30)
{'a': 10, 'b': 20}
```

#### **7.7 update()**

**Updates** the dictionary by **adding or replacing key–value pairs** from another dictionary or iterable.

```python
d1 = {'a': 10, 'b': 20}
d2 = {'b': 200, 'c': 300}
d1.update(d2)
print(d1)
```

**Output:**

```
{'a': 10, 'b': 200, 'c': 300}
```

#### **7.8 copy()**

```python
original = {1: 'geeks', 2: 'for'}
new1 = original
new2 = original.copy()
new1[1]='in'
new2.clear()
print('new2:', new)
print('original:', original)
```

**Output:**

```
new2: {}
original: {1: 'in', 2: 'for'}
```

### **8. Nested Dictionaries**

**Description:**
Dictionaries can contain other dictionaries as values.

**Example 1: Create nested dictionaries**

```python
people = {
    1: {'name': 'John', 'age': '27', 'sex': 'Male'},
    2: {'name': 'Marie', 'age': '22', 'sex': 'Female'}
}
print(people)
```

**Output:**

```
{1: {'name': 'John', 'age': '27', 'sex': 'Male'}, 
 2: {'name': 'Marie', 'age': '22', 'sex': 'Female'}}
```

**Example 2: Accessing nested dictionary**

```python
print(people[1]['name'])
print(people[1]['age'])
print(people[1]['sex'])
```

**Output:**

```
John
27
Male
```

**Example 3: Adding nested dictionary items**

```python
people[3] = {}
people[3]['name'] = 'Luna'
people[3]['age'] = '24'
people[3]['sex'] = 'Female'
people[3]['married'] = 'No'
print(people[3])
```

**Output:**

```
{'name': 'Luna', 'age': '24', 'sex': 'Female', 'married': 'No'}
```

**Example 4: Deleting specific nested keys**

```python
del people[3]['married']
print(people[3])
```

**Output:**

```
{'name': 'Luna', 'age': '24', 'sex': 'Female'}
```

**Example 5: Deleting multiple nested items**

```python
del people[3]
print(people)
```

**Output:**

```
{1: {'name': 'John', 'age': '27', 'sex': 'Male'}, 
 2: {'name': 'Marie', 'age': '22', 'sex': 'Female'}}
```

### **9. Checking Key Existence**

**Description:**
Use the `in` keyword to check if a key exists in a dictionary.

```python
a = {'name': 'John', 'age': 21}
print('name' in a)
print('address' in a)
```

**Output:**

```
True
False
```

### **10. Iterating Through Dictionaries**

**Description:**
Use `for` loops or the `items()` method to iterate over dictionaries.

**Example 1**

```python
num1 = {0: '00', 1: '01', 2: '10', 3: '11'}
for k in num1:
    print(k, ":", num1[k])
```

**Output:**

```
0 : 00
1 : 01
2 : 10
3 : 11
```

**Example 2**

```python
Ages = {'ali': 12, 'mohammad': 20, 'Hamed': 27, 'zahra': 25}
for k, v in Ages.items():
    print(k, v)
```

**Output:**

```
ali 12
mohammad 20
Hamed 27
zahra 25
```

**Example 3**

```python
a = {'name': 'amir', 'age': 10}
for item in a.items():
    print(item)
```

**Output:**

```
('name', 'amir')
('age', 10)
```

## Summary of Dictionary Methods
The table below provides a quick reference to the most commonly used dictionary methods in Python, along with short descriptions of what each one does:

| **Method**                         | **Description**                                    | **Example**                                               |
| ---------------------------------- | -------------------------------------------------- | --------------------------------------------------------- |
| **len(d)**                         | Returns the number of key–value pairs.             | `len({'a':1, 'b':2}) → 2`                                 |
| **d.clear()**                      | Removes all items from the dictionary.             | `d = {'a':1}; d.clear() → {}`                             |
| **d.items()**                      | Returns all key–value pairs as tuples.             | `{'a':1, 'b':2}.items() → dict_items([('a',1), ('b',2)])` |
| **d.keys()**                       | Returns all keys.                                  | `{'a':1, 'b':2}.keys() → dict_keys(['a','b'])`            |
| **d.values()**                     | Returns all values.                                | `{'a':1, 'b':2}.values() → dict_values([1,2])`            |
| **d.pop(key)**                     | Removes a key and returns its value.               | `d = {'a':1}; d.pop('a') → 1`                             |
| **d.popitem()**                    | Removes and returns the last inserted pair.        | `{'a':1, 'b':2}.popitem() → ('b',2)`                      |
| **d.update(other)**                | Adds/replaces items from another dict or iterable. | `d={'a':1}; d.update({'b':2}) → {'a':1,'b':2}`            |
| **d.copy()**                       | Returns a shallow copy of the dictionary.          | `d={'a':1}; c=d.copy() → {'a':1}`                         |
| **d.get(key, default)**            | Returns value or default if key is missing.        | `{'a':1}.get('b', -1) → -1`                               |
| **d.setdefault(key, default)**     | Returns value; adds key if absent.                 | `d={}; d.setdefault('a', 10) → 10`                        |
| **dict.fromkeys(iterable, value)** | Creates dict from keys with same default value.    | `dict.fromkeys(['a','b'], 0) → {'a':0,'b':0}`             |
